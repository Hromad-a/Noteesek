# Noteesek v1.6.0 — release notes

## GitHub release body

**Carousel notes layout** — read one note at a time and swipe left/right between
them, like the onboarding cards. The note fills the whole screen; tap to open it.
The toolbar layout button now cycles **grid → single column → carousel**.

**Safer wipe** — clearing your data now removes only your *own* notes. Notebooks
shared *with* you are left untouched (you can't delete what you don't own), and
the wipe no longer fails when a shared notebook is present.

**Czech onboarding polish** — more natural wording on the welcome screens.

Plus small fixes and refinements.
